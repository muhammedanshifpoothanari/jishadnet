# RishadNet Quick Start Guide

## 🚀 Getting Started

### For New Companies (Registration)

1. **Visit Home Page** → Click "Register Your Company" button
2. **Step 1 - Company Info**
   - Enter company name (required)
   - Select industry (optional)
   - Enter company email (required)
   - Add website (optional)
   - Enter phone number (required)
3. **Step 2 - Admin Account**
   - Enter your full name
   - Enter admin email (different from company email if you prefer)
   - Create strong password (8+ chars, uppercase, lowercase, numbers)
   - Confirm password
   - Optional phone number
4. **Step 3 - Review**
   - Verify all information is correct
   - Accept terms & conditions
   - Click "Complete Registration"
5. **Success** → You'll see "Verification Sent" screen
   - A verification email will be sent (simulated)
   - Click "Go to Login" to proceed

### For Existing Users (Login)

1. **Visit Home Page** → Click "Admin Login"
2. **Select Company** from dropdown (if you manage multiple companies)
3. **Enter Credentials**
   - Email: Your admin email
   - Password: Your password
4. **Click Login**
   - If credentials are valid, you'll be redirected to dashboard
   - If invalid, you'll see an error message

## 📱 Using the Dashboard

### Desktop View
- **Sidebar** (left): Navigation, company selector, user profile, sign out
- **Top Bar** (top): Menu (mobile only), company info, user info, theme toggle
- **Main Content**: Dashboard, devices, alerts, settings

### Mobile View
- **Top Bar** (top): Menu button, logo, theme toggle, notifications
- **Sidebar Menu** (hidden): Swipe or tap menu button to open
- **Bottom Navigation** (bottom): Quick access to main sections
- **Company Selector**: Tap company icon to switch companies

### Managing Companies

**Switch Between Companies**:
1. Open sidebar (mobile) or view directly (desktop)
2. Click on the company selector section
3. Select a different company from the dropdown
4. Dashboard updates automatically
5. You'll see a "Switched to [Company Name]" notification

## 🔐 Security & Best Practices

### Passwords
- Use strong passwords (uppercase + lowercase + numbers)
- Minimum 8 characters
- Don't share your password
- Change password regularly

### Account
- Keep your email updated
- Sign out when finished (especially on shared devices)
- Use only authorized devices

### Data
- Only view data for your assigned company
- Don't share device information unnecessarily
- Report suspicious activity

## 📋 Common Tasks

### Register a Device

1. In dashboard, click "Register Device" (sidebar) or "Register Your Device" (home page)
2. Fill in device information:
   - Your name
   - Your department
   - Device type (laptop, phone, tablet, IoT)
   - MAC address
   - IP addresses
   - Network SSID
   - Optional notes
3. Click "Submit Registration"
4. You'll see confirmation that device is pending approval

### View Devices

1. Click "Devices" in navigation
2. See list of all company devices
3. Click on a device to view details or:
   - Copy MAC address
   - Share via WhatsApp
   - View QR code
   - Download QR code

### Check Alerts

1. Click "Alerts" in navigation
2. View any network alerts or issues
3. Take recommended actions

### Update Settings

1. Click "Settings" in navigation
2. Update company or account information
3. Configure notifications
4. Manage other preferences

## 🆘 Troubleshooting

### Can't Register
- Make sure all required fields are filled (marked with *)
- Check that passwords match in Step 2
- Verify email format is valid
- Check browser console for error details

### Can't Login
- Verify company is selected
- Check email is spelled correctly
- Confirm password is correct
- Make sure the account exists (was registered)
- Check for typos or caps lock

### Can't See Company Data
- Verify correct company is selected (check company selector)
- Check that you're logged in to correct account
- Try switching companies back and forth
- Refresh the page

### Dashboard Looks Wrong
- Clear browser cache and refresh
- Try different theme (light/dark mode)
- Check browser is fully updated
- Try different browser or incognito mode

### Performance Issues
- Close other browser tabs
- Clear browser cache
- Check internet connection
- Restart browser
- Try different network

## 🔄 Session Management

### Your Session
- Automatically created when you login
- Stored in browser (localStorage)
- Persists until you sign out
- Clears if you delete browser data

### Keeping Secure
- Always sign out on shared devices
- Don't leave browser open unattended
- Use private/incognito mode on public computers
- Logout before closing browser if on public device

### On Mobile
- App stores session locally
- Sign out when switching users
- Clear app cache to reset
- Log back in after clearing data
